## Финальные токены (Apple-correct)

### Glass material (4 tiers, grey-tinted)
```css
--glass-ultrathin: rgba(51, 51, 51, 0.18);
--glass-thin: rgba(51, 51, 51, 0.28);
--glass-regular: rgba(51, 51, 51, 0.38);  /* baseline для карт */
--glass-thick: rgba(51, 51, 51, 0.50);
--glass-chrome: rgba(51, 51, 51, 0.42);   /* topbar */
```

### Blur scale
```css
--blur-ultrathin: blur(16px) saturate(140%);
--blur-thin: blur(24px) saturate(160%);
--blur-regular: blur(36px) saturate(180%);
--blur-thick: blur(48px) saturate(200%);
```

### Specular rim (Apple uses uniform top rim, not chromatic)
```css
--spec-rim:
  inset 0 1px 0 rgba(255,255,255,0.4),
  inset 0 -1px 0 rgba(255,255,255,0.06),
  inset 0 0 0 0.5px rgba(255,255,255,0.08);
```

### Ink (tuned for translucent dark glass)
```css
--ink-1: #fff;
--ink-2: rgba(235,235,245,0.85);
--ink-3: rgba(235,235,245,0.66);
--ink-4: rgba(235,235,245,0.46);
```

### Shadows (minimal — iOS 26 flat)
```css
--shadow-card: 0 4px 16px -6px rgba(0,0,0,0.2);
```

### Accent (warm coral, НЕ indigo)
```css
--coral: #FF8A6E;
--amber: #FFC171;
--rose: #FF7D9D;
```

### Radii (iOS 26 concentric)
```css
--r-xl: 28px;  --r-lg: 22px;  --r-md: 16px;  --r-sm: 12px;  --r-pill: 999px;
```

## Фиксы UX

1. **Backdrop-filter "hole"** при смене экрана — Chrome пересоздавал blur layer при `display:none→block`. Фикс: страницы остаются в DOM как `position:absolute + opacity:0 + visibility:hidden`, активная становится `position:relative + opacity:1`. Layer кешируется.
2. **Theme toggle бред** (sun icon в правом углу, текст обрезался "Сменить / тему"). Удалён.
3. **Button text color**: primary button с filled bg → text WHITE (Apple HIG). До этого был `#1a0510` dark на coral = non-Apple.
4. **Removed chromatic rim** (magenta-left, amber-right prismatic edges) — Apple не делает так, uniform white rim.
5. **Removed radial blooms in KPI cards** — Apple uniform glass.
6. **Dropped heavy shadows** (было 24px 64px 80px warm glow) → `0 4px 16px -6px rgba(0,0,0,0.2)` (Apple flat).
7. **Concentric corners + capsule buttons** (Apple iOS 26 signature).
8. **Dual focus ring** (1.5px inner + 5px halo).
